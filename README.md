# Moto G Flash Script

You must  have **adb** and **fastboot** installed on your system.

Tested on Ubuntu  14.04 LTS and Moto G XT1033.

## Usage

```
flash.sh -f <fastboot-path> -r <rom-path> [OPTIONS]
```

## Options

```
-b  Relock Bootloader
```

If anything goes wrong, please register an issue.
